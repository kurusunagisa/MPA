# MPA

MPAは長野高専3Jの授業内で作成した多倍長整数演算ができるプログラムです．Written in C.

## Installation

cloneしてください．

## Usage

Archimedesフォルダ内の`main.c`をgccかclangでコンパイル，実行するといい感じに円周率が計算されます．絶望的に遅いので気を付けて．

パラメータは`mulprec.h`の`#define`を変更してね．
